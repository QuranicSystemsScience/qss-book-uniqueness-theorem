# QSS — Book 3: The Uniqueness Theorem

This repository contains the sources, figures, and reproducibility code for:

**"The Uniqueness Theorem"**  
(Book 3 of the *Quranic Systems Science* project)

---

## 📘 Contents

- `/book/`  
  The manuscript in multiple languages (FR / EN / AR).

- `/figures/`  
  All figures used in the book.

- `/scripts/`  
  Python scripts used to reproduce the experiments and figures.

- `CHANGELOG.md`  
  Version history of the work.

---

## 🔬 Reproducibility

All experiments in this book are:

- Based on explicit scripts,
- Using standard scientific Python libraries,
- Fully reproducible.

The goal is not interpretation, but **structural and computational analysis** of the signal.
